Mattock

A powerful companion to Rake

(current version: 0.2.12)

If Rake won't do it by itself, you oughtta Mattock.

If you survived the pun, you might enjoy this gem.

Features:

* Extensions to Tasklibs to support powerful deerpaths. * A commandline library that supports mocking for tests. * A module to support common templating patterns

Documentation is in a few sections:

  • YARD API documentation
  • RSpec run output
  • Coverage report
